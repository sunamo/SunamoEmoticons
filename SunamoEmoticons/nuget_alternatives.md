# NuGet Alternatives to SunamoEmoticons

This document lists popular NuGet packages that provide similar functionality to SunamoEmoticons.

## Overview

Emoticon and emoji utilities

## Alternative Packages

### Emoji.Net
- **NuGet**: Emoji.Net
- **Purpose**: Emoji detection and manipulation
- **Key Features**: Unicode emoji support, text replacement

### NeoSmart.Unicode
- **NuGet**: NeoSmart.Unicode.Emoji
- **Purpose**: Emoji library
- **Key Features**: Emoji sequences, skin tones, categories

### System.Text.Unicode
- **NuGet**: System.Text.Encodings.Web
- **Purpose**: Unicode utilities
- **Key Features**: Emoji rendering, encoding

## Comparison Notes

Emoji.Net for basic emoji operations. NeoSmart.Unicode for comprehensive emoji support.

## Choosing an Alternative

Consider these alternatives based on your specific needs:
- **Emoji.Net**: Emoji detection and manipulation
- **NeoSmart.Unicode**: Emoji library
- **System.Text.Unicode**: Unicode utilities
